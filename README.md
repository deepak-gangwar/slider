### Installation
```bash
# Clone the repo on you local machine
git clone git@github.com:deepak-gangwar/bundler-webpack-config.git

# To install dependencies
npm install

# To start a dev server locally
npm run dev

# To build the project (output in public folder)
npm run build
```
